# 0.1.4 Release Validation

Date: 2026-07-15

## Scope

Version 0.1.4 makes the distribution's existing license scopes explicit. It adds PEP 639 metadata, a project notice, complete third-party notices, the full CC BY-NC 4.0 text for retained alignment helpers, and expanded source provenance. No inference, decoding, VAD, alignment, output, state, CLI, or Python API logic changed.

The only changed Python file is `src/cohere_transcribe/_version.py`. The source and built-wheel copies of `src/cohere_transcribe/vad/torch_silero.py` both match the 0.1.3 Git revision at SHA-256 `f2e2793d2d62ae38914acbdf7c84ec7ea2879ff36726939dfa7f5e713707c537`.

## License Artifacts

The wheel emits Core Metadata 2.4 and the SPDX expression `Apache-2.0 AND CC-BY-NC-4.0 AND MIT`. The Apache license applies to original project code and documentation. Retained Fairseq MMS/ctc-forced-aligner helpers remain CC BY-NC 4.0. Bundled Silero and faster-whisper components retain their MIT licenses.

The wheel and source distribution include the root Apache license and notice, third-party notices, the complete CC BY-NC 4.0 text, both MIT texts, and the alignment/VAD provenance documents. `THIRD_PARTY_NOTICES.md` records exact source revisions, file boundaries, modifications, asset provenance, and the independent terms of runtime-downloaded models.
